# Insta OSINT CLI

Instagram public profile research toolkit built with Python.

## Features

- Public profile lookup
- Username existence check
- Metadata parsing
- Link extraction
- JSON export

## Installation

```bash
git clone https://github.com/naveen-anon/Insta-osint-CLI-.git
cd insta-osint-cli
pip install -r requirements.txt
```
### Usage ### 

 - Lookup profile
 - python main.py lookup username

 - Export profile
 - python main.py export username

 - Check username
 - python main.py check username
