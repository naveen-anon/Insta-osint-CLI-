# Security Policy

## Supported Versions

The following versions of Insta-OSINT CLI currently receive security updates:

| Version | Supported |
| ------- | ---------- |
| 1.x     | ✅ Yes |
| < 1.0   | ❌ No |

---

## Guidelines

Please include:

- Vulnerability description
- Steps to reproduce
- Potential impact
- Suggested fix (optional)

---

## Responsible Disclosure

Please do not publicly disclose vulnerabilities until they have been reviewed and resolved.

---

## Scope

This project is intended for:

- Public OSINT research
- Educational purposes
- Authorized cybersecurity testing

The following are NOT supported:

- Unauthorized account access
- Credential attacks
- Privacy violations
- Illegal surveillance
- Exploitation of private data

---

## Security Philosophy

Insta-OSINT CLI is designed with an ethical OSINT-first approach.

The project focuses only on publicly accessible Instagram intelligence and metadata analysis.
